# Keyboard Shortcuts

This document lists all keyboard shortcuts available in Pixel Prompt.

## Generation Shortcuts

| Action | Shortcut | Description |
|--------|----------|-------------|
| **Generate Images** | `Ctrl+Enter` (Windows/Linux)<br>`Cmd+Enter` (Mac) | Start image generation with current prompt |
| **Random Prompt** | `Ctrl+R` (Windows/Linux)<br>`Cmd+R` (Mac) | Fill prompt input with random seed prompt (prevents browser reload) |
| **Enhance Prompt** | `Ctrl+E` (Windows/Linux)<br>`Cmd+E` (Mac) | AI-enhance current prompt |

## Download Shortcuts

| Action | Shortcut | Description |
|--------|----------|-------------|
| **Download All** | `Ctrl+Shift+D` (Windows/Linux)<br>`Cmd+Shift+D` (Mac) | Download all completed images from current generation |

## Image Modal

When viewing an image in full-screen modal:

| Action | Shortcut | Description |
|--------|----------|-------------|
| **Close Modal** | `Escape` | Close the image modal |
| **Previous Image** | `Left Arrow` or `←` | Navigate to previous image in generation |
| **Next Image** | `Right Arrow` or `→` | Navigate to next image in generation |

## Utility Shortcuts

| Action | Shortcut | Description |
|--------|----------|-------------|
| **Keyboard Shortcuts Help** | `Ctrl+K` (Windows/Linux)<br>`Cmd+K` (Mac) | Show keyboard shortcuts help dialog |

## Text Input

Standard text editing shortcuts work in the prompt input field:

| Action | Shortcut | Description |
|--------|----------|-------------|
| **Select All** | `Ctrl+A` | Select all text in prompt input |
| **Copy** | `Ctrl+C` | Copy selected text |
| **Paste** | `Ctrl+V` | Paste text from clipboard |
| **Undo** | `Ctrl+Z` | Undo last change |
| **Redo** | `Ctrl+Y` or `Ctrl+Shift+Z` | Redo last undone change |

## Notes

- All `Ctrl` shortcuts use `Cmd` (⌘) on macOS instead
- Shortcuts are automatically disabled when typing in text inputs to avoid conflicts
- Modal shortcuts only work when a modal is open
- Help dialog is platform-aware and shows appropriate modifier keys (Ctrl vs Cmd)
- Press `Escape` to close any open modal or dialog
- Some browser shortcuts may override application shortcuts

## Accessing the Help Dialog

Press `Ctrl+K` (Windows/Linux) or `Cmd+K` (Mac) from anywhere in the application to display an interactive help dialog with all available shortcuts.
