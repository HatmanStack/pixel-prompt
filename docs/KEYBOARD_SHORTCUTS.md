# Keyboard Shortcuts

This document lists all keyboard shortcuts available in Pixel Prompt.

## Global Shortcuts

| Action | Shortcut | Description |
|--------|----------|-------------|
| **Generate Images** | `Ctrl+Enter` (Windows/Linux)<br>`Cmd+Enter` (Mac) | Start image generation with current prompt |
| **Random Prompt** | `Ctrl+R` (Windows/Linux)<br>`Cmd+R` (Mac) | Fill prompt input with random seed prompt |

## Image Modal

When viewing an image in full-screen modal:

| Action | Shortcut | Description |
|--------|----------|-------------|
| **Close Modal** | `Escape` | Close the image modal |
| **Previous Image** | `Left Arrow` or `←` | Navigate to previous image in generation |
| **Next Image** | `Right Arrow` or `→` | Navigate to next image in generation |

## Text Input

Standard text editing shortcuts work in the prompt input field:

| Action | Shortcut | Description |
|--------|----------|-------------|
| **Select All** | `Ctrl+A` | Select all text in prompt input |
| **Copy** | `Ctrl+C` | Copy selected text |
| **Paste** | `Ctrl+V` | Paste text from clipboard |
| **Undo** | `Ctrl+Z` | Undo last change |
| **Redo** | `Ctrl+Y` or `Ctrl+Shift+Z` | Redo last undone change |

## Notes

- All `Ctrl` shortcuts use `Cmd` on macOS instead
- Shortcuts are disabled when typing in text inputs (except text editing shortcuts)
- Modal shortcuts only work when a modal is open
- Some browser shortcuts may override application shortcuts

## Future Enhancements

Potential shortcuts for future implementation:

- `Ctrl+K`: Open keyboard shortcuts help dialog
- `Ctrl+D`: Download focused image
- `Ctrl+Shift+D`: Download all images
- `Ctrl+E`: Enhance current prompt
- Number keys `1-9`: Select preset or focus specific image
- `Tab`: Navigate between UI elements
